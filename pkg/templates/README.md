# 算法模板库 (Algorithm Templates)

这个目录存放经过验证的通用算法代码骨架（Templates）。

## 💎 核心理念：不要重复造轮子

在面试的高压环境下，你不需要现场推导基础算法（如二分查找的边界条件、快排的切分逻辑）。你应该像使用“肌肉记忆”一样，直接写出这些经过千锤百炼的代码骨架，然后专注于解决题目特定的逻辑。

## 📂 目录结构

- **binary_search/**: 二分查找通用模板（解决找目标值、找边界问题）
- **two_pointers/**: 双指针（快慢、对撞）
- **sliding_window/**: 滑动窗口（解决子串/子数组问题）
- **bfs/**: 广度优先搜索（层序遍历、最短路径）
- **dfs/**: 深度优先搜索（回溯、排列组合）

## 🚀 如何使用

1. **理解**: 阅读代码中的注释，理解每一行 **为什么** 这么写（特别是边界条件）。
2. **记忆**: 手写几遍，直到能不看代码默写出骨架。
3. **套用**: 遇到对应类型的题目，先把骨架写出来，再填充逻辑。
