# Go 算法学习路径 (Go Algorithm Roadmap)

## Initial Concept

制定一份对于初学算法的go开发者的算法学习路径，帮助开发者更好的学习算法，每一种数据结构或算法都需要有讲解说明、代码实现、图形解析、性能评估等

---

## 产品愿景
本项目旨在为具有编程经验的 Go 开发者提供一套系统化、实战化的算法学习路径。通过结合详尽的文档说明、高质量的 Go 源码实现、直观的图形解析以及严谨的基准测试，帮助开发者掌握数据结构与算法的核心精髓，提升解决复杂问题的能力，并助力技术面试。

## 目标受众
- **跨语言学习者**：已有其他语言（如 Java, Python, C++）经验，希望通过 Go 语言深度学习算法的开发者。
- **面试求职者**：准备参加 Go 语言相关技术岗位面试，需要巩固算法基础 and 实战能力的开发者。

## 核心模块
1. **基础数据结构**：
   - 线性结构：[已完成] 链表 (Singly/Doubly Linked List)、栈 (Stack)、队列 (Queue/Circular Queue)。
   - 散列结构：哈希表 (Hash Table/Map)。
2. **高级数据结构**：
   - 树形结构：二叉树 (Binary Tree)、二叉搜索树 (BST)、堆 (Heap)。
   - 图论：图的表示与遍历 (Graph)。
   - 特殊结构：并查集 (Union-Find)、布隆过滤器 (Bloom Filter)。
3. **经典算法**：
   - 基础：排序 (Sorting)、搜索 (Searching)、递归 (Recursion)。
   - 进阶：动态规划 (Dynamic Programming)、贪心算法 (Greedy Algorithm)。

## 呈现形式与标准
- **结构化文档**：每个主题包含 Markdown 说明，并使用 Mermaid 或图形化工具进行逻辑解析。
- **代码实现库**：
  - 采用 **混合泛型策略**：基础结构保持简单直观，复杂结构利用 Go 泛型实现类型安全与复用。
  - 包含完整的单元测试（Unit Test）。
- **性能评估体系**：
  - **理论分析**：每个算法均标注时间与空间复杂度 (Big O)。
  - **实测基准**：使用 Go 原生 `testing.B` 提供真实性能数据。
  - **对比分析**：针对同一问题提供不同实现的性能对比视图。
